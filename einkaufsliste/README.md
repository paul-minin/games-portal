# Einkaufsliste (einfach)

Kurze Demo-App mit reiner `HTML/CSS/JS`, speichert lokal im Browser (`localStorage`).

Funktionen:
- Listen erstellen und löschen
- Artikel hinzufügen, abhaken und löschen
- Benutzerdaten werden lokal gespeichert

Anleitung:
1. Öffne `index.html` in deinem Browser (Doppelklick oder per Live Server)
2. Neue Liste erstellen -> Liste auswählen -> Artikel hinzufügen

Tipps:
- Keine Installation nötig.
- Wenn du möchtest, kann ich das Projekt auf React oder als PWA/Flutter erweitern.

Auf dem Handy öffnen (PWA) ✅
- Am einfachsten: Lade die Dateien z. B. auf GitHub Pages oder einen Webserver hoch, öffne die Seite in Chrome (Android) und wähle im Menü „Zum Startbildschirm“. 
- Lokal testen: Starte auf deinem PC einen einfachen Server (z. B. `npx http-server` oder Live Server) und rufe die IP-Adresse vom Handy im Browser auf.
- Hinweis: Für die PWA-Installation ist ein Webserver (https oder localhost) nötig; direktes Öffnen per Datei (file://) unterstützt den Service Worker nicht.

# Spiele-Portal

Dieses Repository enthält ein kleines Portal, das die enthaltenen Spiele (Ordner mit `index.html`) auflistet und in einem Viewer öffnet.

Live: https://paul-minin.github.io/games-portal/

## Hinweise
- Einfache Suche, Sortierung und Einbettung via `iframe`.
- Inhalte können in `games.json` verwaltet werden.
